**关键技术决策（research.md）**：

| 决策 | 结论 |
|---|---|
| G 进度计算 | 纯函数 + Celery 异步刷新，不实时重算 |
| 四步法会话 | 数据库持久化（JSONB），支持草稿 |
| 行动推荐 | v1 规则引擎，v2 考虑 Claude API |
| 多租户隔离 | PostgreSQL 独立 Schema（Bridge 模式）|
| BI 看板缓存 | Redis 5min TTL，导入后主动失效 |
| 角色权限 | JWT 携带角色，FastAPI 依赖注入层校验 |
| 导入触发刷新 | Celery 任务链：import_task → ogsm_refresh_task |
| 前端状态 | Pinia 分模块（ogsm/analysis/dashboard）|
