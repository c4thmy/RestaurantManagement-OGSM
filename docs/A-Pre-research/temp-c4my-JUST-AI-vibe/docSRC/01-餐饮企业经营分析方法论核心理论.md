# 餐饮企业经营分析方法论核心理论

**基于《如何通过经营分析做好经营改善》课程提炼**

---

## 一、核心定义：3C经营分析框架

### 1.1 经营分析的本质

用数据信息，向内对表、向外对标，看清自己、看清顾客、看清竞争，发现异动、拆解问题、锁定归因、解决问题，组织持续提升改善与复制盈利的闭环管理动作。

### 1.2 核心价值

将餐饮经营中零散的成功经验通过数据化锚定、结构化拆解、标准化落地，转化为可复制、可考核、可动态迭代的企业标准化经营体系，让成功从依赖个人经验的偶然结果，变为能全域规模化落地的必然成果。

---

## 二、经营分析三大层面

### 2.1 宏观层面：行业与环境（PEST分析）

**核心目的**：判断行业趋势、机会与风险，为战略方向提供依据

- **政策(P)**：食品安全、环保法规
- **经济(E)**：人均收入、行业增速
- **社会(S)**：消费观念、生活方式
- **技术(T)**：智慧餐厅、AI应用

### 2.2 中观层面：赛道与竞争

**核心目的**：找到市场位置，识别竞争对手，制定差异化竞争策略

- **细分赛道**：品类规模、生命周期
- **竞争格局**：竞对分析、市场份额
- **产业链**：上下游议价能力、配套

### 2.3 微观层面：单店与运营

**核心目的**：发现运营问题，提升单店盈利与效率，为复制提供依据

- **产品**：核心贡献、菜单结构
- **顾客**：画像分析、会员运营
- **运营**：坪效人效、流程优化
- **模型**：成本结构、利润分析

## 三、经营分析四大困境与解决方案

### 3.1 困境一：只有财务报表，没有运营分析

**问题表现**：报表显示毛利率下降，但不知道是采购价格高、损耗大、投料不准还是销售结构变化；开会时运营、厨房、采购互相扯皮，问题不了了之；数据颗粒度过大，缺乏分析逻辑和方法，无法将数据转化为决策依据。

**解决方案**：建立多层级指标体系，从财务数据深挖到运营细节

### 3.2 困境二：拉新不理想，老顾客复购上不去

**问题表现**：沉迷于品牌有知名度的自我感觉，忽视顾客真实反馈；盲目跟风做流量推广，放弃会员复购，活动一停门店立刻冷清；老顾客流失严重，只能归咎于市场竞争太激烈。

**解决方案**：建立用户视角，从顾客需求出发做精准营销和会员运营

### 3.3 困境三：生意突然变差，后知后觉

**问题表现**：附近新开店抢走30%客流，月底看报表才发现；直接竞对升级产品换菜单，压根不关注；社区食堂靠便宜现炒取胜，被截流却不自知。

**解决方案**：建立外部视角，定期竞对分析和市场监测

### 3.4 困境四：不会设计盈利模型，成功没法复制

**问题表现**：母店生意好就多开店，全凭感觉这个地段不错；每家店投资额都不一样，认为不同的店投资就应该不一样；决策凭感觉，成败靠运气。

**解决方案**：建立模型视角，在决策前进行盈利设计、沙盘推演和风险测算

## 四、经营分析四步法（核心方法论）

### 4.1 第一步：一比目标

**口诀**：一眼看穿总盘，锁定异常波动

**操作**：拉出预算/同比/环比三张表，标红差异率>5%的指标，快速锁定问题

**对比维度**：

| 时间维度 | 对比方式 | 看什么 |
|---------|---------|--------|
| 同店环比 | 短期经营波动 | 上月到这月发生了什么 |
| 同店同比 | 剔除季节因素 | 看真实增长趋势 |
| 定基比 | 品牌生命周期 | 门店是在成长还是在衰退 |

| 空间维度 | 对比方式 | 看什么 |
|---------|---------|--------|
| 同销售额、不同门店对比 | 经营管理效率 | 同样的收入谁的成本更低、利润更高 |
| 同商圈、不同门店对比 | 排除外部因素 | 看内部管理差距 |
| 标杆店 vs 问题店 | 找出可复制的成功经验 | |

### 4.2 第二步：二拆数据

**口诀**：层层剥开洋葱，定位问题源头

**操作**：按区域/店型/时段/渠道下钻，将整体异动拆解为单点异动，精准定位

**拆解路径示例**：
1. 渠道拆解：缺口在哪个渠道（堂食 vs 外卖）
2. 因素拆解：缺口来自客流还是客单
3. 时段拆解：缺口集中在哪个时段（工作日/周末/午市/晚市）
4. 来源拆解：缺口来自哪类顾客（自然到店/平台引流/会员复购）

### 4.3 第三步：三定归因

**口诀**：深挖五问为什么，分清偶发系统

**操作**：运用5Why分析法，判定是执行偏差还是模型失效，找到根本原因

**归因公式**：
- 收入异动 = 客单因素 + 客流因素
- 客单因素 = (本期客单 - 同期客单) × 同期客流
- 客流因素 = (本期客流 - 同期客流) × 本期客单

### 4.4 第四步：四做决策

**口诀**：对策责任到人，闭环跟踪结果

**操作**：输出行动清单，明确谁(Who)在何时(When)做什么(What)，确保闭环，并跟踪复盘，成功经验固化为标准体系

**决策三类型**：
- **市场渗透型（抢流量）**：适用于新店、新品牌或市场占有率低的门店
- **成本领先型（降成本）**：适用于供应链有优化空间、人效坪效偏低、费用冗余的企业
- **价值提升型（提客单）**：适用于客群消费潜力未充分挖掘、产品结构有待优化的老店

## 五、3C经营分析指标体系全览

### 5.1 自身分析指标（看自己做得怎么样）

| 指标类别 | 核心指标                                  |
| ---- | ------------------------------------- |
| 收入   | 总营收、新老店、渠道、时段、商圈、会员/非会员收入、系列/单菜品销售额   |
| 成本率  | 标准成本、理论成本、实际成本、原材料损益率、食材成本率、库存周转率、损耗率 |
| 毛利率  | 综合毛利率、系列/单菜品毛利率、TOP20%毛利额贡献           |
| 费用率  | 空间费用（租售比、坪效）、人员费用（人工成本率、人效）、摊销费用、能源费用 |
| 净利率  | 利润总额、利润率、投资回报率、投资回报周期                 |
| 运营效率 | 翻台率、出餐时长、等位时长、就餐时长、时段损耗率、暗访得分率        |

### 5.2 顾客分析指标（看用户要什么）

| 指标类别 | 核心指标 |
|---------|---------|
| 客流量 | 总客流、渠道、时段、商圈、会员/非会员订单 |
| 客单价 | 平均客单、渠道、时段、商圈、会员/非会员客单、场景人数单均 |
| 用户资产 | 会员总数、复购率、留存率、新客占比、私域会员、客诉率、大众点评满意度 |
| 市场与营销 | 各渠道引流效果、活动参与率、活动ROI、新客获客成本 |
| 顾客需求与空白 | 目标客群画像、消费场景偏好、未满足痛点、需求趋势 |

### 5.3 竞争分析指标（看对手在做什么）

| 指标类别 | 核心指标 |
|---------|---------|
| 品类赛道 | 品类市场规模、增长率、头部品牌市占率、细分赛道热度、品类生命周期 |
| 竞对360度 | 竞对品牌定位、客单价、门店模型、产品结构、核心爆款、促销活动、用户评价 |
| 产品结构差异 | 产品结构、价格结构、需求结构（你卖什么/对手卖什么/顾客想要什么） |
| 差异化定位 | 品牌核心价值主张、差异化卖点、目标客单锚点、选品组合策略 |
| 模型验证与迭代 | 新店盈利模型、选址模型验证、MOT触点满意度、数据反馈循环效率 |

## 六、盈利模型设计方法论

### 6.1 盈利模型核心指标前提

- 利润率 > 15%
- 租售比 < 15%
- 人工 < 25%
- 坪效 > 100元/平米
- 人效 > 1000元/人/天
- 日上座次数 > 2次
- 投资回收周期 < 租期的2/3

### 6.2 盈利模型推演步骤

**步骤一：先找北斗星**——明确模型的目标和边界（利润率目标、投资回收周期、人效/坪效/租售比目标）

**步骤二：定义变量与常量**——客流量区间（变量A）、日均上座次数（变量B）、单平米日租金（变量C）、客单价/毛利率/单个餐位面积等（关键常量）

**步骤三：核心公式推导**
- 理想面积 = 日客流量 / 上座次数 × 单个餐位面积
- 实际座位数 = 理想面积 / 单个餐位面积
- 用工人数 = 店面积 / 人均服务面积

**步骤四：输出动态模型**——三档客流做压力测算（乐观/正常/悲观），验证利润和投资回收周期是否达标

## 七、方法论总结

餐饮企业经营分析的核心是建立**数据化经营驱动组织持续迭代优化、复制成功**的能力。

**核心要点**：
1. 经营分析不是为了分析而分析，而是为了发现问题、解决问题、复制成功
2. 数据拆解要遵循MECE原则，不重不漏
3. 归因分析要深挖5个为什么，分清偶发和系统性问题
4. 决策要因时制宜、因店制宜，解决当下主要矛盾
5. 盈利模型是从计算升级为动态推演，让模型决策该开多大的店
6. 客流和客单的分析要回归顾客需求和市场竞争本质
